How to run the Trading Risk Engine web app:

1. Download both files:
   - trading_risk_app_index.html
   - app.js
   Place them in the same folder.

2. Open the HTML file:
   - Double-click it, OR
   - Run a local server:
     python -m http.server 8000
     Then open http://localhost:8000/trading_risk_app_index.html

3. Use the sidebar to:
   - Add systems  
   - Add instruments  
   - Create new trades  
   - Track PnL  

4. Data is stored locally in your browser.

5. Use Export / Backup / Restore options to save your data permanently.
